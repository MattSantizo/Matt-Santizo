Arduino

For this project my partner and I had to make something that is interactive with an Arduino board, bread board, USB cable, and coding. So my parter an I put a button, LED lights, buzzer, resistors, and jumper wires. When the USB is plugged into the Arduino board and computer, the bread board now has electricity. Once it gets electricity the buzzer turns on right away and when the button is pushed the LED lights turn on, but once you let go of the button the LED lights turn off. The way the coding works is in the code you tell it which slots in the arduino board you want electricity to flow through. The electricy then goes throught the jumper wires, then the resistor, then the button, then the LED lights, and lastly back to the Arduino board. The buzzer works the same way it just doesnt go through the button.
